== Description ==

This is a OIS Bizcraft Checkout Payment Plus Gateway for WooCommerce.

With Integrated OIS Bizcraft Setup (Easy Setup), connecting to OIS Bizcraft is as simple as filling your OIS Bizcraft merchant info - no complicated API keys to cut and paste.
